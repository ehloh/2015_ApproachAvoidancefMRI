Materials for the project ‘Parsing hippocampal contributions to approach-avoidance conflict’, accepted for publication in Cerebral Cortex in 2016. Project is outlined in Summary pdf. 

Code for data acquisition, analysis of behavioural data (model-based and model-agnostic), and fMRI analysis are included here. 

Authors: Eleanor Loh, Zeb Kurth-Nelson, David Berron, Peter Dayan, Emrah Duzel, Ray Dolan and Marc Guitart-Masip.
